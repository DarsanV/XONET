# XONET Frontend

React app (Vite + TanStack Start + Tailwind).

## Structure

```
public/           static files
src/
  components/     UI + layout
  hooks/
  lib/            utils & data
  routes/         pages
  index.css       global styles
  router.tsx
  start.ts
```

## Commands

- `npm run dev` — development
- `npm run build` — production build
- `npm run preview` — preview build
- `npm run lint` — ESLint

Node 20+ required.
